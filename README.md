# maven-release

#### 介绍
airlenet版本仓库


#### 使用说明

github 版本仓库
```xml
    <repositories>
        <repository>
            <id>airlenet-releases</id>
            <name>Airlenet Releases</name>
            <url>https://airlenet.github.io/maven-release</url>
            <snapshots>
                <enabled>false</enabled>
            </snapshots>
        </repository>				
    </repositories>
```


gitee 版本仓库
```xml
    <repositories>
        <repository>
            <id>airlenet-releases</id>
            <name>Airlenet Releases</name>
            <url>https://airlenet.gitee.io/maven-release</url>
            <snapshots>
                <enabled>false</enabled>
            </snapshots>
        </repository>				
    </repositories>
```
